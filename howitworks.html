Sky Screenshot Stats

How It Works

The program extracts the creation date from all the screenshots. These dates can then be used to generate some statistics. This data is retrieved by one of the following 2 ways:

  • The program first looks to the pictures' file names, to see if these contain the creation date.
  • If the file name doesn't contain a valid date, then the program looks to the creation date of the file, inside the file's properties.

Note: Sometimes the creation date of a file gets overwritten in the file's properties, thus causing it to become invalid. One example of where this might happen, is when the user opens the image, makes changes to it and saves it.


What happens when it isn't possible to retrieve a valid creation date for a screenshot?

In such cases, where neither the creation date in the properties of the file is valid, nor the file name contains a valid date.

The file will be ignored, and won't be used to generate the statistics. The page that displays the statistics, also has a metric that shows how many files were processed, and how many were invalid.
